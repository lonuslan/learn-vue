属于响应式的数组方法：
push() pop() shift() unshift() splice() sort() reverse()
  • {{item}}
在 console 使用 app.infos.push('e') 时，数据会立刻发生变化，这就是相应式的方法
push方法,从数组最后面追加元素,可变参数 ...num

通过index下标改变数组的值，不是响应式

pop()方法，删除最后的一个元素

shift()方法，删除数组的第一个元素

unshift()方法, 在数组的最前面追加元素, 可变参数

splice()方法：删除/插入/替换元素：
spalice(1): 删除第1个元素后的所有元素；a b c d---> a
splice(1, 2): 从第1个元素开始，删除其后面连续的2个元素； a b c d----> a d
splice(1, 2, 'm', 'n'): 相当于做完 splice(1, 2) 之后，又向后面追加了两个元素； a b c d ---> a m n d
splice(1, 0, 'm', 'n'): 相当于做完 （splice(1, 0) -> 从第一个位置后面删除0个元素） 之后，追加了两个元素：a b c d ---> a m n b c d

sort()方法：排序

reverse()方法：数组内容反转
